SQL es un lenguaje de consulta estructurado que se utiliza para interactuar con bases de datos relacionales. Fue creado por IBM en la década de 70 y se ha convertido en el lenguaje de base de datos más popular del mundo. SQL se utiliza para crear, modificar, eliminar y consultar datos en bases de datos relacionales.
